### Tunnel de conversion

```
Visite → Inscription → Démo → Appel → Proposition → Signature
  1000      100         50      20        15          10
```

**Taux de conversion cibles :**
- Visite → Inscription : 10%
- Inscription → Démo : 50%
- Démo → Appel : 40%
- Appel → Proposition : 75%
- Proposition → Signature : 67%

### Process de vente

#### Étape 1 : Qualification (5 min)
```
Questions clés :
- Quel est votre rôle ?
- Comment gérez-vous vos données actuellement ?
- Quel est votre budget analytics ?
- Avez-vous un data scientist ?
- Timeline de décision ?
```

#### Étape 2 : Démonstration (20 min)
```
Structure :
1. Contexte client (3 min)
2. Import données live (5 min)
3. Analyse IA en temps réel (5 min)
4. Dashboard personnalisé (5 min)
5. Q&A (2 min)
```

#### Étape 3 : Proposition (24h)
```
Inclure :
- Récapitulatif besoins
- Solution proposée
- Prix et remises
- ROI estimé
- Timeline d'implémentation
- Garantie satisfaction (30 jours)
```
