## Introduction

**InsightGov Africa** est une plateforme SaaS qui transforme automatiquement vos données en dashboards intelligents grâce à l'intelligence artificielle. Conçue spécifiquement pour les organisations africaines (Ministères, ONG, Entreprises), elle permet de :

- 📊 **Visualiser** vos données en quelques clics
- 🤖 **Analyser** automatiquement vos indicateurs clés
- 📈 **Suivre** vos KPIs en temps réel
- 📤 **Exporter** des rapports professionnels
- 🔗 **Partager** vos analyses avec vos équipes

---

## Démarrage rapide

### 1. Créer un compte

1. Rendez-vous sur [insightgov.africa](https://insightgov.africa)
2. Cliquez sur **"Créer un compte"**
3. Remplissez vos informations :
   - Email professionnel
   - Mot de passe (8 caractères minimum, avec majuscule, minuscule et chiffre)
   - Nom complet
4. Complétez les informations de votre organisation :
   - Nom de l'organisation
   - Type (Ministère, ONG, Entreprise, etc.)
   - Secteur d'activité
   - Pays

### 2. Mode Démonstration

Pour tester la plateforme sans créer de compte :
1. Cliquez sur **"Essayer la démo"**
2. Explorez toutes les fonctionnalités avec des données fictives
3. Créez votre compte quand vous êtes prêt

## Importer vos données

### Formats acceptés

| Format | Extension | Taille max |
|--------|-----------|------------|
| CSV | .csv | 50 MB |
| Excel | .xls, .xlsx | 50 MB |

### Comment importer

1. **Depuis la page d'accueil**, cliquez sur **"Importer un fichier"**
2. **Glissez-déposez** votre fichier ou cliquez pour parcourir
3. **Attendez** le traitement automatique
4. **Consultez** l'aperçu de vos données

### Structure recommandée

Pour de meilleurs résultats, votre fichier devrait contenir :

```
✅ En-têtes de colonnes clairs (première ligne)
✅ Données homogènes par colonne
✅ Dates au format JJ/MM/AAAA ou YYYY-MM-DD
✅ Nombres sans symboles de devise dans les cellules
✅ Pas de lignes vides au début du fichier
```

### Génération de données de démo

Pas de données sous la main ? Utilisez notre générateur :

1. Sélectionnez un **secteur** (Santé, Éducation, Finance, Agriculture)
2. Cliquez sur **"Générer des données de démo"**
3. Un dataset réaliste est créé instantanément

## Analyse IA

### Comment ça fonctionne

Une fois vos données importées, notre IA (GPT-4o) :

1. **Détecte automatiquement** les types de colonnes
   - Nombres, dates, catégories, identifiants
2. **Identifie les KPIs pertinents** pour votre secteur
3. **Génère des visualisations** adaptées
4. **Fournit des insights** et recommandations

### Types d'analyses

| Type | Description |
|------|-------------|
| **Tendances** | Évolution dans le temps |
| **Comparaisons** | Entre catégories/périodes |
| **Distributions** | Répartition des valeurs |
| **Corrélations** | Relations entre variables |

### Personnaliser l'analyse

Vous pouvez affiner les suggestions de l'IA :
1. Modifiez les types de colonnes détectés
2. Ajoutez des descriptions contextuelles
3. Sélectionnez les KPIs prioritaires

## Explorer les dashboards

### Vue d'ensemble

Le dashboard principal affiche :
- 📊 **KPIs clés** avec tendances
- 📈 **Graphiques interactifs**
- 📋 **Données brutes** navigables

### Onglets disponibles

| Onglet | Contenu |
|--------|---------|
| **Vue d'ensemble** | Résumé des indicateurs principaux |
| **KPIs** | Tous les indicateurs détaillés |
| **Graphiques** | Visualisations personnalisables |
| **Données** | Tableau avec filtres |

### Filtres dynamiques

1. Cliquez sur **"Ajouter un filtre"**
2. Sélectionnez une colonne
3. Choisissez la condition (égal, supérieur, contient...)
4. Appliquez

### Filtres rapides

Des filtres prédéfinis sont disponibles :
- **Période** : Ce mois, ce trimestre, cette année
- **Région** : Par zone géographique
- **Statut** : Actif/inactif, validé/en attente

## Exporter et partager

### Export PDF

1. Cliquez sur **"Exporter"** → **"PDF"**
2. Configurez les options :
   - Titre du rapport
   - Logo de votre organisation
   - Pages à inclure
3. Cliquez sur **"Générer"**
4. Téléchargez le fichier

### Export Excel

1. Cliquez sur **"Exporter"** → **"Excel"**
2. Choisissez les données à inclure :
   - Données filtrées ou complètes
   - Avec ou sans calculs
3. Téléchargez le fichier `.xlsx`

### Partage par lien

1. Cliquez sur **"Partager"**
2. Générez un lien public
3. Définissez la durée de validité (1 jour à 1 an)
4. Copiez et envoyez le lien

**Note** : Le lien fonctionne même pour les personnes sans compte.

## FAQ

### Général

**Q: Mes données sont-elles sécurisées ?**
R: Oui. Vos données sont isolées par organisation, chiffrées en transit et au repos. Nous respectons le RGPD.

**Q: Puis-je utiliser la plateforme hors ligne ?**
R: Non, InsightGov Africa nécessite une connexion internet pour l'analyse IA.

**Q: Quels navigateurs sont supportés ?**
R: Chrome, Firefox, Safari, Edge (dernières versions).

### Import de données

**Q: Pourquoi mon fichier est rejeté ?**
R: Vérifiez que :
- Le format est CSV, XLS ou XLSX
- La taille est inférieure à 50 MB
- Le fichier n'est pas corrompu

**Q: Puis-je importer plusieurs fichiers ?**
R: Oui, selon votre plan. Chaque fichier crée un dataset distinct.

### Analyse IA

**Q: Combien de temps prend l'analyse ?**
R: De quelques secondes à 2 minutes selon la taille du fichier.

**Q: L'IA peut-elle se tromper ?**
R: L'IA suggère des analyses basées sur les données. Vérifiez toujours la pertinence pour votre contexte.

### Paiement

**Q: Quels modes de paiement sont acceptés ?**
R: Paystack accepte cartes bancaires, Mobile Money (Orange, MTN, Wave), et virements.

**Q: Puis-je payer en Franc CFA ?**
R: Oui, Paystack convertit automatiquement.
